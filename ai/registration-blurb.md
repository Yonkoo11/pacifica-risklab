Perp parameter stress tester. Feed it a Pacifica market, pick a historical crash (Oct '25 flash crash, LUNA, JELLY incident), and it simulates what would've happened — how many positions get liquidated, whether funding rates spike past caps, how deep the cascade goes. Then you tweak the parameters (lower max leverage, tighten maintenance margin) and run it again to see what survives. Built on Pacifica's API using live market data.
